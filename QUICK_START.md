# Бърз старт за новите функционалности

## 🎯 Какво е ново?

1. **Интерактивна карта** за избор на локация при създаване на сигнал
2. **Динамични категории** - управление от dashboard
3. **Отговорни лица** по райони и категории
4. **Автоматично назначаване** на сигнали към отговорни лица

## 🚀 Стъпки за стартиране

### 1. Миграция на базата данни

```bash
# Създаване на миграция
npm run prisma:migrate

# Когато бъдете попитани за име, използвайте:
# add_categories_and_responsible_persons
```

### 2. Seed на начални данни

```bash
# Зареждане на категории и примерни отговорни лица
npx tsx prisma/seed-categories.ts
```

Това ще създаде:
- 16 категории (дупки, улично осветление, боклук, графити и др.)
- Примерни отговорни лица за първите 3 района
- Поддръжка за всички 24 района на София

### 3. Стартиране на приложението

```bash
npm run dev
```

### 4. Тестване на новите функции

#### Като администратор:

1. **Управление на категории**
   - Отидете на `http://localhost:3000/admin/categories`
   - Създайте нова категория или редактирайте съществуваща
   - Активирайте/деактивирайте категории

2. **Управление на отговорни лица**
   - Отидете на `http://localhost:3000/admin/responsible-persons`
   - Добавете отговорни лица за различни райони и категории
   - Филтрирайте по район или категория

#### Като гражданин:

1. **Създаване на сигнал с карта**
   - Отидете на `http://localhost:3000/dashboard/new-report`
   - Кликнете "Избери от карта"
   - Изберете локация на картата
   - Системата автоматично ще определи района
   - Изберете категория от новия списък
   - Създайте сигнала

2. **Проверка на автоматично назначаване**
   - След създаване на сигнала, проверете дали е назначен към отговорно лице
   - В детайлите на сигнала ще видите информация за назначеното лице

## 🔧 Конфигурация

### Добавяне на отговорни лица

За всеки район и категория можете да добавите отговорни лица:

1. Име и фамилия
2. Email
3. Телефон
4. Длъжност (по избор)
5. Район
6. Категория

### Автоматично назначаване

Когато гражданин създаде сигнал:

```
Сигнал → Район (от координати) → Категория → Отговорно лице
```

Системата намира първото активно отговорно лице за комбинацията район + категория.

## 🐛 Отстраняване на проблеми

### Проблем: Миграцията не работи

```bash
# Reset на базата данни (ВНИМАНИЕ: Изтрива всички данни!)
npx prisma migrate reset

# След това отново:
npm run prisma:migrate
npx tsx prisma/seed-categories.ts
```

### Проблем: Картата не се зарежда

Проверете дали Leaflet CSS е зареден правилно. Компонентът използва dynamic import с `ssr: false`.

### Проблем: Не се определя район

Уверете се, че координатите са в границите на София:
- Latitude: 42.6 - 42.8
- Longitude: 23.1 - 23.5
